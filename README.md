# 🐍 Snake Game (Python Turtle)

A classic Snake Game built using Python's Turtle graphics module with clean Object-Oriented Programming (OOP) structure.

---

## 🚀 Features
- Smooth snake movement
- Food spawning at random locations
- Score tracking system
- Collision detection:
  - Wall collision
  - Self collision
- Modular code structure (separate files for each component)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
- Classes & Objects
- Event Listeners
- Game Loop
- Collision Detection
- Python Turtle Graphics

---

## 📂 Project Structure
├── main.py # Game loop & controls
├── snake.py # Snake movement logic
├── food.py # Food spawning system
├── scoreboard.py # Score tracking & display


---

## 🎮 Controls
- ⬆️ Up Arrow → Move Up  
- ⬇️ Down Arrow → Move Down  
- ⬅️ Left Arrow → Move Left  
- ➡️ Right Arrow → Move Right  

---

## 🖥️ How to Run
1. Clone the repository:
2. Navigate to the project folder
3. Run the game


---

## 📸 Gameplay
- Eat food to grow the snake 🟥
- Avoid hitting walls or yourself ❌
- Try to get the highest score 🎯
